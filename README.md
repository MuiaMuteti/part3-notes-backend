# Notes Backend

A simple REST API for managing notes, built in Part 3 of the [Fullstack Open](https://fullstackopen.com/) course.

## Setup

 **Clone** the repo and install dependencies:

   ```bash
   git clone https://github.com/MuiaMuteti/part3-notes-backend.git
   cd part3-notes-backend
   npm install
   npm run dev
